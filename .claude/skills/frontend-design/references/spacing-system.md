# Spacing System

## Base Unit

4px base. Multiples: 4, 8, 12, 16, 20, 24, 32, 48, 64, 96, 128.

## CSS Custom Properties

```css
--space-1: 0.25rem;   /* 4px */
--space-2: 0.5rem;    /* 8px */
--space-3: 0.75rem;   /* 12px */
--space-4: 1rem;      /* 16px */
--space-5: 1.25rem;   /* 20px */
--space-6: 1.5rem;    /* 24px */
--space-8: 2rem;      /* 32px */
--space-12: 3rem;     /* 48px */
--space-16: 4rem;     /* 64px */
--space-24: 6rem;     /* 96px */
--space-32: 8rem;     /* 128px */
```

## Rules

- Start with too much whitespace, then tighten â€” it's easier to remove space than to add it
- **Spacing encodes relationship**: label-to-field gap < field-to-field gap < section-to-section gap. Close = related, far = distinct.
